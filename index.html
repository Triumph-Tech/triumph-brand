triumph.tech / brand

Triumph Brand Kit

This repository is the single source of truth for Triumph Tech's brand assets, colors, typography, and voice. It is designed to be consumed by AI agents and automation tools to produce consistently branded work.

Agent Instructions

Start with brand-kit.json — it contains all structured brand data in a single machine-readable file. For tone, logo usage, and writing rules, read brand-guidelines.md. All asset paths in the JSON are relative to the repo root. When producing branded output, use Invictus Blue (#0A2540) as the primary color, Goldman Sans as the sole typeface, and follow the voice summary in the voice object. The logo files are SVGs in logos/ — prefer the combination mark in most contexts.

brand-kit.json

JSON Schema

Key Description
brand Company name, tagline ("Diligent Innovation"), website URL, and brand initiatives including Digital Revival.
colors.primary Invictus Blue (#0A2540) — main brand color. Includes RGB, CMYK, text_on value, and 3 tonal variations.
colors.secondary Electric Blue (#4BA9DF) — accents and highlights. Includes RGB, CMYK, text_on value, and 3 tonal variations.
colors.accent Twilight Blue (#5D758B) — backgrounds and UI elements. Includes RGB, CMYK, text_on value, and 3 tonal variations.
colors.grayscale 8-step grayscale scale from #FFFFFF to #1A1A1A.
colors.semantic Success, warning, error, and info colors. To be defined.
typography.primary Goldman Sans — sole authorized typeface. Lists all font weights with self-hosted file paths. Default weight: 400.
logos Paths for combination_mark, logo_mark, wordmark, and digital_revival logos in light and dark variants. Includes clearspace rule and approved backgrounds.
mascot Magnus — name, description, and asset file paths.
voice Brand voice summary, tone descriptors, words to avoid, and example phrases.
usage Dos and don'ts for brand application.
brand_elements.image_wash Layer specs for the Invictus Blue image treatment: desaturate image to 0%, set opacity to 15% with Luminosity blend, place Invictus Blue beneath.
